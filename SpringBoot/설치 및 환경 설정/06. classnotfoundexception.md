# [classnotfoundexception]
```
문제 상황

라이브러리가 중간에 바껴서, 꼬인 것 같은 느낌.
버전이 달랐거나, 없는걸 추가해서 그렇거나
```

<br>

```java
// 전

dependencies {
  ...
	implementation 'com.mysql:mysql-connector-java'
	...
}
```
```java
// 후
dependencies {
  ...
	implementation group: 'com.mysql', name: 'mysql-connector-j', version: '8.0.33'
	...
}
```
